## 如何贡献
CodingTools 项目非常欢迎贡献者。 最简单的贡献方式，就是参与讨论，并向我们提出产品的改进意见。此外，我们也非常欢迎测试和提交BUG。

如果你希望提交代码，请通过 Pull Request 的方式提交，并确保你的代码符合以下要求：

- 具有详细的功能说明
- 具有完整的单元测试
- 具有充分的注释
- 我会在收到请求后，及时处理提交。
